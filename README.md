# Coin Collector (金幣收集者)

這是一個使用 Python 和 Pygame 開發的簡單趣味小遊戲。玩家需要控制角色在從天而降的金幣雨中收集金幣，並在有限的時間內挑戰最高分！

## 🎮 遊戲特色

*   **角色選擇系統**：遊戲開始前可以選擇三種不同的英雄角色（Cat, Cute, Ghost），每個角色都有獨特的造型。
*   **流暢的操作體驗**：支援左右移動與跳躍，手感順暢。
*   **倒數計時挑戰**：在 45 秒內盡可能收集更多金幣。
*   **趣味評語系統**：遊戲結束後，根據您的得分給予不同的評價（如：「你好菜」、「還行」、「不錯」）。
*   **互動式結算畫面**：支援滑鼠點擊選擇「繼續玩」或「關閉遊戲」。

## 🛠️ 安裝需求

請確保您的電腦已安裝 Python。此外，您需要安裝 `pygame` 函式庫。

```bash
pip install pygame
```

## 🚀 如何執行

1.  確認您已下載專案所有檔案（包含 `assets` 資料夾內的圖片）。
2.  在終端機 (Terminal) 中執行以下指令：

```bash
python 惟媗gemini.py
```

## 🕹️ 操作說明

### 角色選擇畫面
*   **左右方向鍵**：切換角色
*   **Enter 鍵**：確認選擇
*   **滑鼠點擊**：直接點擊角色亦可選擇

### 遊戲進行中
*   **左右方向鍵 (或 A/D)**：左右移動
*   **空白鍵 (Space) / 上方向鍵**：跳躍

### 結算畫面
*   **滑鼠左鍵**：點擊按鈕選擇重玩或離開

## 📝 遊戲規則

1.  遊戲時間共 **45 秒**。
2.  每收集一枚金幣可獲得 **10 分**。
3.  接住越多金幣，評價越高！

## 📂 專案結構

*   `惟媗gemini.py`: 遊戲主要程式碼
*   `assets/`: 存放遊戲圖片資源 (角色圖、金幣圖等)
